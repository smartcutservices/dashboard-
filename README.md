# Smart Cut Services Dashboard

Depot autonome du dashboard admin Smart Cut Services.

## Point d'entree
- `dashboard.html`
- `index.html` redirige automatiquement vers `dashboard.html`

## Contenu
- dashboards admin principaux
- modules vendeurs et impression
- garde d'acces admin
- configuration Firebase commune au dashboard

## Remarque
Ce depot est une extraction du dashboard depuis le projet principal afin de le faire evoluer de maniere independante.
